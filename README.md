# Spotifycli

> List new album releases via Spotify

## Install

$ bundle

$ Add env variables to ~/.bashrc

```
export CLIENT_ID=''
export CLIENT_SECRET=''
```

$ source ~/.bashrc

Make executable

$ chmod +x spotifycli.rb

$ mv spotifycli.rb spotifycli

$ sudo mv spotifycli /usr/local/bin/

$ spotifycli


## Usage

$ spotifycli help discover
